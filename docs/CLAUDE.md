# 문서 작성 규칙

## 문장 단위 줄바꿈

마크다운 문단에서 **한 문장이 끝나면 반드시 빈 줄로 줄바꿈**한다.
한 문단에 여러 문장을 이어 붙이지 않는다.

### 왜

- 시각적 호흡을 주어 읽기 쉽게 만들기 위해
- 유저가 일관되게 선호하는 스타일

### 어디에 적용

- 본문 문단 — 문장 사이에 빈 줄 하나
- Tip / Info / Warning 박스 안의 본문도 동일
- 표 셀 안에서 두 문장 이상을 쓸 때는 `<br>` 로 줄바꿈

### 예외

- 리스트 항목 한 줄 안의 짧은 설명 (한 문장짜리 bullet 은 그대로)
- 코드 블록, 인용문
