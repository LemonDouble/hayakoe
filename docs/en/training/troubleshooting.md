# Troubleshooting

If you encounter problems during training, conversion, or deployment, please file an issue on [GitHub Issues](https://github.com/LemonDouble/hayakoe/issues).

Including the following information helps us reproduce and resolve the issue faster.

- Which step it occurred at (Data Prep GUI / Preprocessing / Training / Quality Report / Publish)
- Full error message + stack trace
- GPU / CUDA version / OS
- Steps to reproduce
